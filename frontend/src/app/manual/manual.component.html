Hosting, c'est quoi ?

Le service d'hébergement "MiNET Hosting" permet aux adhérents de MiNET de créer des Machines Virtuelles et de les stocker dans les serveurs MiNET.

Une VM, c'est quoi ?

Une machine virtuelle, ou « virtual machine », est « le client » créé dans un environnement informatique, « l’hôte ». Plusieurs machines virtuelles peuvent coexister sur un seul hôte. Les principaux fichiers qui constituent une machine virtuelle sont un fichier journal, un fichier de paramètres de RAM non volatile, un fichier de disque virtuel et un fichier de configuration - VMware

Une VM, pourquoi ?

Une VM peut servir d'espace de stockage, de bac à sable pour des tests de développement, de plateforme d'hébergement de site web... Les possibilitées sont multiples !

Une VM, comment ?

Le service MiNET Hosting vous offre la possbilité de créer une ou des VMs en quelques clics.
Vous devez tout d'abord vous identifier avec vos IDs MiNET dans la section Login.

Dans l'onglet "Your VMs", vous pouvez procéder à la création en spécifiant notamment le nom de votre VM, si vous voulez une authentification par clé SSH ou non, et le nom d'utilisateur de la VM (évitez root pour ne pas avoir de conflits) ainsi que le mot de passe.

La VM créée dispose de spécificités techniques qui dépendent du template utilisé. Il est possible de changer ces paramètres sous présentation d'un justificatif auprès de l'équipe MiNET.

Les deux templates disponibles sont :

  • Bare VM : crée une VM que vous pouvez manier à votre convenance. Il s'agit d'un modèle de base composé de openssh, fail2ban, et les utilitaires debian par défaut.

Ce template est composé de 2 GO de Ram, 2 coeurs CPU virtuels et 10 Go de stockage.

  • Static web server : déploie une VM avec un serveur web (Wordpress) prêt à l'emploi.

Ce template est composé de 1 GO de Ram, 1 coeur CPU virtuel et 3 Go de stockage.

Une fois votre formulaire de création de VM validé, il vous faudra attendre environ 3 minutes selon le template souhaité pour que la création de la VM s'effectue.

Vous aurez ensuite accès à votre VM dans l'onglet "Your VMs".
L'IP de votre VM sera en 157.159.195.X

Attention : Il n'est possible de créer que 2 VM par adhérent.

Après la création, que faire ?

Vous pouvez dans un premier temps vous connecter sur votre VM. Il existe deux manières :

  • via votre terminal en exécutant la commande
    ssh <nomd'utilisateur>@<ipdevotrevm>
    Puis en entrant votre mot de passe.
  • via l'utilitaire webssh intégré sur notre site qui vous permet de vous connecter directement sur votre navigateur. Il vous faudra aussi entrer l'IP de votre machine et les identifiants que vous avez entré lors de la création.

Une entrée DNS, pour quoi faire ?

Le DNS permet d’associer un nom compréhensible, à une adresse IP. On associe donc une adresse logique, le nom de domaine, à une adresse physique l’adresse IP. - Nameshield

Dans l'onglet MiNET DNS entries, vous pouvez choisir d'ajouter une adresse à votre service web en nomdevotresite.h.minet.net

Il vous suffit d'entrer le nom textuel que vous souhaitez obtenir et l'IP de votre serveur.
Vous pourrez ensuite accéder à votre site via son entrée DNS.
Exemple : toto.h.minet.net
